fn to_accessible(&self) -> Result<Self::Accessible, Self::Error>

Creates an Self::Accessible from the existing accessible item.

Errors

This may fail based on the implementation of. Generally, it fails if the accessible item does not implement to accessible interface. This shouldn’t be possible, but this function may fail for other reasons. For example, to convert a zbus::Proxy into a Self::Accessible, it may fail to create the new crate::accessible::AccessibleProxyBlocking.
