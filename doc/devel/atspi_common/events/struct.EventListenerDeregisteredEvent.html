Struct atspi_common::events::EventListenerDeregisteredEvent

source · 
pub struct EventListenerDeregisteredEvent {
    pub item: ObjectRef,
    pub deregistered_event: EventListeners,
}
Expand description

An event that is emitted by the registry daemon, to inform that an event has been deregistered to no longer listen for.

Fields§

§item: ObjectRef

The ObjectRef the event applies to.

§deregistered_event: EventListeners

A list of events that have been deregistered via the registry interface. See atspi-connection.

A static match rule string for DBus. This should usually be a string that looks like this: "type='signal',interface='org.a11y.atspi.Event.Object',member='PropertyChange'"; This should be deprecated in favour of composing the string from Self::DBUS_MEMBER and Self::DBUS_INTERFACE.
