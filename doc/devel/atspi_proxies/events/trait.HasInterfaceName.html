pub trait HasInterfaceName {
    const DBUS_INTERFACE: &'static str;
}
Expand description

A specific trait only to define an interface name. This is useful for event wrappers like ObjectEvents, which, while it does not have other information required to implement the BusProperties trait, you can indeed attach in interface name for all sub events of ObjectEvents.

This trait is not object-safe.

Required Associated Constants§

source

const DBUS_INTERFACE: &'static str

A static interface string for DBus. This should usually be a string that looks like this: "org.a11y.atspi.Event.*";

This trait is not object safe.
