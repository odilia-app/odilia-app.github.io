Expand description

A helper trait which is (usually) implemented on CacheItem, generically over [GerProperty::P]. P must implement PropertyType, which defines the representation returned from GetProperty::get_property in the EventProp<P> generic.

TryFromState is auto-implemented for any EventProp<P> and GetProperty<P> for CacheItem. In practical terms, this means you can use it like an extractor:

// in `main.rs`
use odilia::tower::{EventProp, Name};
async fn handle_event(
    EventProp(name): EventProp<Name>,
) {
    todo!()
}
