Expand description

A Predicate tests if a value satisfies a particular refinement type.

Used in conjunction with Refinement.

§Example

use refinement::Predicate;

struct LessThanTen;

impl Predicate<i32> for LessThanTen {
    fn test(x: &i32) -> bool {
        *x < 10
    }
}
